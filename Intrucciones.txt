Intrucciones para abrir la pagina:
-Crear el ambiente virtual dentro de la carpeta principal con //python -m venv env
-Activar el ambiente virtual con el archivo activate que se situa en env/Scripts
-Luego volver a la ruta principal para instalar los requerimentos con //pip install -r requirements.txt
-Abrir el programa con py manage.py runserver
